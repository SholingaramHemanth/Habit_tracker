## ✨ Features

### ⚙️ Core User Settings (The 12-Category System)
Our flagship feature is a granular, hierarchical settings engine that gives you absolute control over your environment:

1.  **👤 Profile Customization**: High-resolution avatars, meaningful personal goals, and identity initialization.
2.  **⏰ Reminder & Notifications**: Precision-timed check-ins with AI-optimized notification slots.
3.  **🌙 Theme Engine**: Seamless transitions between Dark (Void), Light (Ethereal), and System modes.
4.  **📅 Week Start Preference**: Global support for Sunday or Monday starts.
5.  **🔁 Habit Reset Rules**: Choose your difficulty—Strict (1 miss resets), Forgiving (2 misses), or Infinite Flow.
6.  **📊 Units & Measurement**: Metric, Imperial, and custom units for hydration, distance, and time.
7.  **🎮 Gamification Toggles**: Control the visibility of XP, Levels, and Power Ranks.
8.  **🔕 Vacation Mode**: Pause your journey without losing your streak during exams or travels.
9.  **🔐 Data Privacy**: Biometric app lock, notification masking, and local-first encryption.
10. **📤 Research & Export**: Export your entire life's data as CSV or PDF for deep analysis.
11. **🛠️ Advanced AI Suggestions**: Predictive habit prompts based on your energy levels and behavior.
12. **🧪 Beta Features**: Early access to experimental Aura tools before they hit the main branch.

## 🔔 Aura Intelligence Notification System
The notification system is designed to be persistent but non-intrusive, featuring:
- **Pending Habit Alerts**: Real-time reminders when your habit momentum is at risk.
- **Milestone Achievements**: Cinematic celebrations for 7, 30, and 100-day streaks.
- **Quick Action Protocol**: Mark habits as 'Ready' or 'Snooze' directly from the notification panel without leaving your workflow.
- **Haptic Feedback**: Subtle 'Shake' animations on the bell icon to signify unread intelligence.

## 🛠️ Technical Manifesto

### The Stack
- **Engine**: React 18 + TypeScript (Strict Mode)
- **Styling**: Tailwind CSS 4.0 + Custom CSS Variable Injection
- **Animations**: Framer Motion / Motion (Physics-based springs)
- **VFX**: Canvas Confetti + Mesh Gradient Engines
- **State**: Persistent LocalStorage with Deep-Merge Initialization

### Architecture
Aura follows a modular, component-driven architecture:
- `src/components/dashboard`: High-fidelity UI elements (Sidebar, StatsGrid, HabitList).
- `src/pages`: Cinematic page containers (Dashboard, Auth, Landing).
- `src/context`: Global Aura environment state (Theme, Accessibility).
- `src/types`: Strongly-typed interfaces ensuring data integrity across the stack.

## 🚀 Installation & Deployment

### Prerequisites
- Node.js (v18.0.0 or higher)
- npm or yarn

### Local Setup
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/SholingaramHemanth/Habit_tracker.git
    cd Habit_tracker
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Launch the Aura Engine**:
    ```bash
    npm run dev
    ```

4.  **Production Build**:
    ```bash
    npm run build
    ```

## 🎨 Design Philosophy: The Aura Aesthetic
Aura utilizes a custom-built **Glassmorphic Design System**. Every element is built with:
- `Backdrop Blur (20px+)`: Creating depth and focus.
- `Translucent Borders`: Subtly defining boundaries without breaking immersion.
- `Dynamic Mesh Gradients`: Evolving backgrounds that reflect the time of day.
- `Isomorphic Iconography`: Using Lucide-React refined with custom CSS filters.
